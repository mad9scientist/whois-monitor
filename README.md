# Whois Monitor
Monitor the whois records for domains with alerts

Requires: PHP 8.2, Database (SQLite/MariaDB/MySQL)

## Uses
* Laravel 11.x
* Jetstream
* Livewire

## Features

* Monitor Domain Whois Data (via RDAP)
  * Dates
    * Expiration, Updated, Creation
  * Nameservers
  * Domain Status
  * Registry
  * Contact Information
    * Admin
    * Registrant
    * Tech
* Alert via
  * Email (default)
  * Discord/Google Chat
